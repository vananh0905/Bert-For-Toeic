### 2. How to use
  **Prerequisite**: python3.6+
  - Clone project and navigate to the project folder
  - Create virtual environment (optional, recommended)
   > ```virtualenv venv```
   > ```source venv/bin/activate```
  - Install requirements
   > ```pip install requirements.txt```
  - Run main.py
   >```python main.py```
